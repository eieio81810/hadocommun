# ハドこみゅ (Hadocommun)

HADOコミュニティ「ハドこみゅ」のObsidian Vaultです。

## 📚 このリポジトリについて

このリポジトリは、ハドこみゅプロジェクトのドキュメントおよびObsidian Vaultを管理しています。

## 📁 コンテンツ

- **[[Welcome|Welcome.md]]** - ハドこみゅの紹介とコミュニティ情報
- **Markdown_knowhow.md** - Markdownの書き方ガイド
- **Contributing.md** - コントリビューションガイド
- **Our_activities.md** - 活動紹介
- **.obsidian/** - Obsidian vault設定

## 🚀 Obsidianで開く

1. Obsidianアプリを開く
2. 「Vaultを開く」を選択
3. このリポジトリのルートディレクトリを選択

## 🔌 プラグインについて

ハドこみゅプラグイン「NexusPM」は別リポジトリで管理されています：

- **NexusPM**: [https://github.com/eieio81810/nexuspm](https://github.com/eieio81810/nexuspm)

### NexusPMのインストール

1. [NexusPMリリースページ](https://github.com/eieio81810/nexuspm/releases)から `main.js` と `manifest.json` をダウンロード
2. このvaultの `.obsidian/plugins/nexuspm/` フォルダにコピー
3. Obsidianの設定でプラグインを有効化

## ✏️ 編集について

- すべてのMarkdownファイルはUTF-8（BOMなし）で保存してください
- Obsidian記法に従ってドキュメントを編集できます
- リンク、タグ、埋め込みなどObsidianの機能がすべて使用可能です

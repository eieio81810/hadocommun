## Obsidianとは？

**Obsidian** は、マークダウン形式でノートを作成・管理できるアプリケーションです。

ハドこみゅでは、Obsidianを使って：

- 📚 HADOの技術・戦術を体系的に整理
- 🔗 情報同士を **リンク** でつなぎ、ネットワーク構造で管理
- 👥 複数メンバーで知識を共有・更新

しています。

### グラフビュー

Obsidianの**グラフビュー**では、ドキュメント同士のリンク関係が視覚化されます。

これにより：
- どの情報がどこにつながっているか一目でわかる
- 新しい発見や関連性に気づける
- 知識がニューラルネットワークのように有機的に成長

---

## なぜ Git + Obsidian？

### Git を使う理由

- ✅ **バージョン管理**: 変更履歴をすべて記録
- ✅ **共同編集**: 複数人で同時に作業可能
- ✅ **復元可能**: いつでも過去の状態に戻せる

### Obsidian を使う理由

- ✅ **マークダウン形式**: シンプルで読みやすい
- ✅ **リンク機能**: 情報を網目状につなげられる
- ✅ **プラグイン**: カスタマイズで機能拡張

この組み合わせで、**オープンで進化し続ける知識ベース** を作っています！

## 寄稿の流れ

### 1. リポジトリをクローン

```bash
git clone https://github.com/eieio81810/hadocommun.git
```

### 2. Obsidianでフォルダを開く

- Obsidianアプリを起動
- 「Vaultを開く」→ `obsidian_hadocommun` フォルダを選択

### 3. ドキュメントを作成・編集

マークダウン記法でノートを作成します。  
詳しくは [[Markdown_knowhow|マークダウン書き方ガイド]] をご覧ください。

### 4. 内部リンクでつなぐ

他のノートと関連付けるため、`[[ノート名]]` 形式でリンクを作成します。

**例:**
```markdown
[[Welcome|トップページ]]に戻る
HADOの[[基本戦術]]を学ぶ
```

### 5. Gitでコミット＆プッシュ

```bash
git add .
git commit -m "新しいノートを追加"
git push
```

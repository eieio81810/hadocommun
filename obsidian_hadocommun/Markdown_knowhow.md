# マークダウン書き方ガイド

オブシディアンはマークダウンという書き方で文章を書きます。

---

## 見出し

`#` の数で見出しのレベルを設定します。

```markdown
# 見出し1（大）
## 見出し2
### 見出し3
#### 見出し4
```

---

## 文字装飾

### 太字
```markdown
**太字にしたい文字**
```
**太字にしたい文字**

### 斜体
```markdown
*斜体にしたい文字*
```
*斜体にしたい文字*

### 取り消し線
```markdown
~~取り消したい文字~~
```
~~取り消したい文字~~

### ハイライト
```markdown
==重要な文字==
```
==重要な文字==

---

## リスト

### 箇条書き
```markdown
- 項目1
- 項目2
  - 項目2-1（インデント）
  - 項目2-2
- 項目3
```

- 項目1
- 項目2
  - 項目2-1（インデント）
  - 項目2-2
- 項目3

### 番号付きリスト
```markdown
1. 最初
2. 次
3. 最後
```

1. 最初
2. 次
3. 最後

---

## チェックリスト

```markdown
- [ ] 未完了のタスク
- [x] 完了したタスク
```

- [ ] 未完了のタスク
- [x] 完了したタスク

---

## リンク

### 外部リンク
```markdown
[表示テキスト](https://example.com)
```

### 内部リンク
```markdown
[[別のノート名]]
[[ノート名#見出し]]
```

---

## 引用

```markdown
> 引用したい文章
> 複数行も可能
```

> 引用したい文章
> 複数行も可能

---

## コード

### インラインコード
```markdown
これは `コード` です
```
これは `コード` です

### コードブロック
````markdown
```python
def hello():
    print("Hello")
```
````

```python
def hello():
    print("Hello")
```
